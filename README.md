# Mobile Reception Menu (Prototype)

Простой мобильный прототип QR-меню для переговорок:
- новости компании;
- заказ напитков;
- вызов технического специалиста;
- мини-игра в стиле Dino.

## Стек
- `HTML`
- `CSS`
- `Vanilla JavaScript`

## Запуск
1. Откройте `index.html` в браузере.
2. Для проверки конкретной переговорки можно добавить query-параметр:
   - `index.html?room=Orion`
   - `index.html?room=Atlas`

## Что уже есть
- mobile-first интерфейс;
- меню из 4 сценариев;
- симуляция отправки заказов/заявок (сохранение последних событий в `localStorage`);
- мини-игра Dino на `canvas` с сохранением рекорда в `localStorage`.
